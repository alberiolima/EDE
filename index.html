Welcome to EDE

EDE is a full blown editor of the server file that is responsible for the drop from monsters in a MMORPG game Perfect World.

Edit&Create

EDE is not just an editor, it can also create new files from scratch.

100% Correct

EDE will preserve the floating point precision and all other data of the original file. Saving over original file or as new will produce identical file to the byte. EDE is also able to correct some of the errors of the original file as well.

Additional Info

EDE provides additional info that will immensely help adjusting the drop rates of the items.
